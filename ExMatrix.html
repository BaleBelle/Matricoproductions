คำอธิบายภาษา C

ที่ใช้สร้างฟังก์ชันสำหรับบวกเมทริกซ์ 2x2 และทดสอบเวลาที่ใช้ในการทำงานของฟังก์ชันดังกล่าว

โค้ดประกอบด้วยฟังก์ชัน addMatrices ที่ใช้สำหรับการบวกเมทริกซ์ และ main ซึ่งเป็นฟังก์ชันหลักที่ใช้เรียกใช้ addMatrices และวัดเวลาที่ใช้ในการทำงานของฟังก์ชันนี้ด้วยฟังก์ชัน clock() ในไลบรารี time.h โดยมีรายละเอียดดังนี้:

  • ฟังก์ชัน addMatrices:
    • รับพารามิเตอร์เข้ามา 3 ตัวคือ A, B, และ result ทั้งหมดเป็นเมทริกซ์ 2x2 ของจำนวนเต็ม.
    • วนลูปสองรอบเพื่อบวกเมทริกซ์ A และ B โดยใส่ผลลัพธ์ลงในเมทริกซ์ result.
  • ฟังก์ชัน main:
    • สร้างตัวแปร A, B และ result เป็นเมทริกซ์ 2x2 ของจำนวนเต็มและกำหนดค่าเริ่มต้นให้กับ A และ B.
    • วัดเวลาที่ใช้ในการทำงานของฟังก์ชัน addMatrices โดยใช้ฟังก์ชัน clock() เพื่อคำนวณเวลาเริ่มต้นและสิ้นสุดของการทำงาน.
    • พิมพ์ผลลัพธ์ของเมทริกซ์ result.
    • พิมพ์เวลาที่ใช้ในการทำงานของฟังก์ชัน addMatrices.

โดยทั่วไปแล้วโปรแกรมนี้มีจุดประสงค์ในการสอนการใช้งานฟังก์ชันในภาษา C และวิธีการวัดเวลาที่ใช้ในการประมวลผลโดยใช้ฟังก์ชัน clock()

ฟังก์ชัน AddMatrices: รับพารามิเตอร์เข้ามา 3 ตัวคือ A, B, และ result ทั้งหมดเป็นเมทริกซ์ 2x2 ของจำนวนเต็ม. วนลูปสองรอบเพื่อบวกเมทริกซ์ A และ B โดยใส่ผลลัพธ์ลงในเมทริกซ์ result. ฟังก์ชัน Main: สร้างตัวแปร A, B และ result เป็นเมทริกซ์ 2x2 ของจำนวนเต็มและกำหนดค่าเริ่มต้นให้กับ A และ B. สร้างตัวแปร stopwatch เพื่อวัดเวลาที่ใช้ในการทำงานของฟังก์ชัน AddMatrices. เริ่มต้นตรวจวัดเวลาด้วย Stopwatch.Start() ก่อนที่จะเรียกใช้ AddMatrices และสิ้นสุดการตรวจวัดเวลาด้วย Stopwatch.Stop() หลังจากที่เสร็จสิ้นการทำงาน. พิมพ์ผลลัพธ์ของเมทริกซ์ result ที่ได้จากการบวก. พิมพ์เวลาที่ใช้ในการทำงานของฟังก์ชัน AddMatrices โดยใช้ stopwatch.Elapsed.TotalSeconds.
